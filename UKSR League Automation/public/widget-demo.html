The UKSimRacing Championship Widget allows you to display live championship standings on any website. Choose from multiple embedding options to fit your needs.

  • Customizable appearance
  • Auto-refresh capabilities

üîó Embedding Options

üìÑ Method 1: iframe Embed (Simplest)

Copy and paste this HTML directly into any webpage:

Basic Usage

<iframe src="https://your-domain.com/championship-widget?leagueId=12345&seasonId=67890" 
        width="100%" 
        height="600" 
        frameborder="0">
</iframe>

Enhanced View with Custom Options

<iframe src="https://your-domain.com/championship-widget?leagueId=12345&seasonId=67890&viewType=enhanced&division=Pro&maxRows=10" 
        width="100%" 
        height="600" 
        frameborder="0">
</iframe>

Detailed Race-by-Race Breakdown

<iframe src="https://your-domain.com/championship-widget?leagueId=12345&seasonId=67890&viewType=detailed&maxRows=15" 
        width="100%" 
        height="700" 
        frameborder="0">
</iframe>

‚ö° Method 2: JavaScript Widget (More Flexible)

Include the script and add data attributes for more integration options:

Basic JavaScript Widget

<script src="https://your-domain.com/championship-widget.js"></script>

<div id="uksimracing-championship" 
     data-league-id="12345" 
     data-season-id="67890">
</div>

Enhanced View with Multiple Options

<script src="https://your-domain.com/championship-widget.js"></script>

<div id="uksimracing-championship" 
     data-league-id="12345" 
     data-season-id="67890"
     data-view-type="enhanced"
     data-division="Overall"
     data-max-rows="10"
     data-show-gap="true"
     data-auto-refresh="300">
</div>

Statistics View Example

<script src="https://your-domain.com/championship-widget.js"></script>

<div id="stats-widget" 
     data-league-id="12345" 
     data-season-id="67890"
     data-view-type="stats"
     data-division="Pro">
</div>

üöÄ Advanced Usage

Multiple Widgets on One Page

<!-- Pro Division --> <div id="championship-pro" data-league-id="12345" data-season-id="67890" data-division="Pro" data-max-rows="10"> </div> <!-- Amateur Division --> <div id="championship-am" data-league-id="12345" data-season-id="67890" data-division="Amateur" data-max-rows="10"> </div>

Styling Customization

<style> .uksimracing-widget { border: 3px solid your-brand-color; border-radius: 15px; } .uksimracing-header { background: your-brand-gradient !important; } </style>
